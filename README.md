## 📌 **Overview**

**DROS PRO** is a professional-grade stress testing tool for web applications. It features multi-threaded attack capabilities, proxy rotation, and real-time statistics. Designed for **educational purposes and authorized security testing only**.

> ⚠️ **IMPORTANT**: This tool is for **YOUR OWN SERVERS** or with **WRITTEN PERMISSION** only. Unauthorized use is **ILLEGAL**.

---

## ✨ **Features**

| Feature | Description |
|:-------:|:------------|
| 🚀 **Multi-Threaded** | Support up to 500 concurrent threads |
| 🔄 **Proxy Rotator** | Automatic proxy cycling to avoid IP bans |
| 📊 **Real-time Stats** | Live RPS counter and progress bar |
| 🎯 **HTTP/HTTPS** | Full support for both protocols |
| ⚡ **GET/POST** | Both HTTP methods supported |
| 📈 **Status Tracking** | Real-time status code monitoring |
| 🛡️ **Anti-Block** | Random User-Agent rotation |
| 💾 **Proxy Validation** | Test proxies before attack |
